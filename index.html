SELFTRACKER

Privacy Policy

Overview

Self Tracker is a Chrome browser extension that helps users understand their browsing habits by tracking time spent on websites and categorizing activity into Work, Personal, and Entertainment. We are deeply committed to protecting your privacy.

The short version: We collect nothing. All your data stays on your device. There is no server, no account, and no tracking by us whatsoever.

What Data Is Collected

Self Tracker collects the following data locally on your device:

  • Website URLs and domain names — to identify which sites you visit.
  • Page titles — to display readable site names in your dashboard.
  • Time spent on each site — measured by tracking active tab duration.
  • Visit timestamps — to organize your activity by day and hour.
  • Category assignments — whether a site is classified as Work, Personal, or Entertainment.
  • User category overrides — when you manually reclassify a site.

Where Data Is Stored

All data is stored exclusively in your browser's local storage using the Chrome Storage API (chrome.storage.local). This means:

  • Your data never leaves your device.
  • Your data is not transmitted to any server, cloud service, or third party.
  • Your data is not accessible to the extension developer or anyone else.
  • Your data is tied to your local Chrome profile and is not synced across devices.

Data Sharing

We do not share, sell, transfer, or disclose any user data to any third party. There is no analytics service, no telemetry, no crash reporting, and no external API calls that transmit user data.

Data Retention

Browsing data is retained locally for up to 90 days. Data older than 90 days is automatically deleted by the extension. You can also manually delete all data at any time from the extension's dashboard using the "Clear All Data" button.

Third-Party Services

The extension loads the Chart.js library from cdn.jsdelivr.net to render visual charts in the dashboard. This is a widely used, open-source JavaScript library. No user data is sent to this CDN — it is used solely to load the charting code. The extension also loads Google Favicons (google.com/s2/favicons) to display website icons. No browsing data is transmitted in these requests beyond the domain name for the favicon image.

Permissions Used

  • tabs — To detect the active website and page title for time tracking.
  • activeTab — To read the current tab information when the user opens the popup.
  • storage — To save tracking data locally on the device.
  • alarms — To periodically save active tab data every 30 seconds.
  • host permissions (all URLs) — To read URLs from all websites for categorization purposes.

User Control

You have full control over your data at all times:

  • You can view all tracked data in the extension dashboard.
  • You can change the category of any website if the auto-detection was incorrect.
  • You can export your data as a CSV file at any time.
  • You can delete all data instantly from the dashboard.
  • You can uninstall the extension at any time, which removes all stored data.

Children's Privacy

Self Tracker is not directed at children under the age of 13. We do not knowingly collect any personal information from children.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date below. Since the extension does not collect any contact information, users are encouraged to review this page periodically.

Contact

If you have any questions or concerns about this privacy policy or the Self Tracker extension, please open an issue on our GitHub repository or contact us at the email listed on the Chrome Web Store listing.
